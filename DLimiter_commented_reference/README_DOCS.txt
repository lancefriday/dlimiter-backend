DLimiter (commented reference set)

This folder holds a documented reference implementation of the DLimiter features built in your project:
- Web login + register + logout (session based)
- Dashboard with counts: owned files, total links created, active links
- Files page: upload, list, delete, create share links (public or restricted)
- Links page: list links, copy download URL, revoke link
- Download page: token resolution, expiry + max downloads, restricted access by email
- Download events logging (audit)
- Admin: users list + toggle admin, download events table

Use these files for documentation-style comments and explanations. Align names/paths with your current repo.
